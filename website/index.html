Quick Start

Get up and running in 3 commands

1

Install Templates

dotnet new install NSharp.Templates
2

Create Project

dotnet new nsharp-console -o MyApp
3

Run

cd MyApp && dotnet run

Why N#?

⚡

Simple Syntax

Go-inspired syntax with no semicolons, short variable declarations with :=, and convention-based visibility.

🔗

Full .NET Interop

Perfect interoperability with C# and the entire .NET ecosystem. Use any NuGet package seamlessly.

🎯

Modern Type System

Discriminated unions, structural typing, and pattern matching that C# lacks. Pragmatic approach to nullability.

🛠️

Great Tooling

VS Code extension with IntelliSense, syntax highlighting, and full MSBuild integration.
